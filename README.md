# iAngelo Battle Quest

**iAngelo Quest Studio presenta:** **iAngelo Battle Quest - Beta Fundadora**, primer battle royale web del estudio.

Beta ligera en espanol para navegador. Mantiene Node.js, Colyseus, PM2, licencia MIT y notas open source. Produccion usa puerto 2568, modo PM2 fork, una instancia y max_memory_restart 450M.

## Vision

Entra a la arena, sobrevive, elimina rivales y domina el ranking. El tono visual es epico/fantasy-tech con obsidiana, dorado, fuego y cian.

## Estado beta

- Cliente HTML5 con Phaser 3 y React.
- Servidor Node.js con Colyseus.
- Textos visibles en espanol.
